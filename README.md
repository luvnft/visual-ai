# visual-ai
2024 Tiktok TechJam

## Problem Statement
- **Track 7: On Unleashing Potential in Machine Learning Infrastructure**

## Requirements

- Backend:
  - Python 3.11.4
  - pip install -r requirements.txt
- Frontend:
  - cd frontend
  - npm install
  - npm start
- For testing:
  - Firefox browser

## Dev Tools Used

- IDE: IntelliJ IDEA 2024.1.4 (Ultimate Edition)
- Version Control: Git, Github
- Package Managers: npm, pip
